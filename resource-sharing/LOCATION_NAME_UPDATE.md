### 3. 智能匹配最近地点

**自动定位逻辑：**
```typescript
// 1. 获取用户GPS坐标
const lat = position.coords.latitude;
const lng = position.coords.longitude;

// 2. 计算与所有预设地点的距离
let closestLocation = presetLocations[0];
let minDistance = Infinity;

presetLocations.forEach(loc => {
  const distance = calculateDistance(lat, lng, loc.lat, loc.lng);
  if (distance < minDistance) {
    minDistance = distance;
    closestLocation = loc;
  }
});

// 3. 如果在100公里内，使用最近的预设地点名称
const locationData = minDistance < 100
  ? closestLocation
  : { name: '当前定位位置', lat, lng };
```

**使用Haversine公式计算距离：**
```typescript
function calculateDistance(lat1, lng1, lat2, lng2): number {
  const R = 6371; // 地球半径
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLng = (lng2 - lng1) * Math.PI / 180;
  const a =
    Math.sin(dLat / 2) * Math.sin(dLat / 2) +
    Math.cos(lat1 * Math.PI / 180) *
    Math.cos(lat2 * Math.PI / 180) *
    Math.sin(dLng / 2) *
    Math.sin(dLng / 2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
  return R * c;
}
```

## 🎯 使用流程

### 方式一：自动定位
1. 点击"自动定位我的位置"按钮
2. 浏览器获取GPS坐标
3. 系统自动匹配最近的预设地点
4. 显示地点名称（如：海南大学）

### 方式二：快速选择
1. 点击"手动设置"按钮
2. 打开手动设置模态框
3. 从列表中选择预设地点
4. 点击确认，位置更新

### 方式三：自定义输入
1. 点击"手动设置"按钮
2. 打开手动设置模态框
3. 在输入框中输入地点名称
4. 点击确认或按Enter键
5. 位置更新为自定义名称

## 🔧 技术实现

### 数据缓存

```typescript
// 保存位置到localStorage
localStorage.setItem('userLocation', JSON.stringify({
  name: '海南大学',
  lat: 20.0440,
  lng: 110.3190,
  timestamp: Date.now(),
}));

// 加载缓存位置
const savedLocation = localStorage.getItem('userLocation');
if (savedLocation) {
  const parsed = JSON.parse(savedLocation);
  // 检查是否过期（10分钟）
  if (Date.now() - parsed.timestamp < 600000) {
    setCurrentLocation(parsed);
  }
}
```

### 距离计算阈值

- **匹配阈值**：100公里
- **在阈值内**：使用最近的预设地点名称
- **超出阈值**：显示"当前定位位置"

### 优先级规则

1. **手动设置** > **自动定位** > **缓存位置**
2. 手动设置具有最高优先级
3. 缓存位置有效期10分钟
